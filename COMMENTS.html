Steps to prepare environment to run the code
  1. Download the Oracle JDK version 8 by clicking here
  2. Add JAVA_HOME to your environment variables pointing to where JAVA is installed
  3. Unzip the proto.zip file
  4. Download and install the apache maven product Here
  5. Open a unix shell
  6. Check and make sure that the proto directory holds the txnlog.dat file
  7. Run the proto.sh shell script which should run the program
Once the program runs the following will happen...
  1. The program will read the proto/txnlog.dat file
  2. If the file is missing it will inform the user and exit with code 1
  3. It will read the data.
  4. It will process the records based on the specifications as I understood them
  5. It will generate and display two web pages using the default application for an html file
    • If there's no default application for the html file type then you will have to check the temp directory for two new html files.
      1. proto...html
      2. Final%20Report...html
    • There is information on how to set a default browser by clicking here
  6. It will attempt to launch those html files using the default application browser
